## Kbd (Keyboard)

Press <Kbd>Cmd</Kbd> + <Kbd>K</Kbd> to open the command palette.

Use <Kbd>Ctrl</Kbd> + <Kbd>C</Kbd> to copy and <Kbd>Ctrl</Kbd> + <Kbd>V</Kbd> to paste.

## Usage

Import the components you need at the top of your MDX file:

```mdx
import Callout from '../../components/mdx/Callout.astro';
import Badge from '../../components/mdx/Badge.astro';
```

Then use them anywhere in your content.
