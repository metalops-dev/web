# metalops.dev

A minimal brutalist technical blog built with [Astro](https://astro.build).

## Quick Start

```bash
pnpm install
pnpm dev          # Start dev server
pnpm build        # Build for production
pnpm test         # Run tests
```

## Features

- Dark/light mode toggle
- MDX support with custom components
- Automated test suite for data validation
- Blog with series support, tags, and archives
- SEO optimized with OG images and JSON-LD

## Project Structure

```
src/
├── components/     # Astro & MDX components
├── content/blog/   # Blog posts (Markdown/MDX)
├── data/           # YAML-inspired JS data files
├── layouts/        # Page layouts
├── pages/          # Routes
└── styles/         # Global CSS
```
